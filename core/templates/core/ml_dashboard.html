Machine Learning Modelle für Gewichtsvorhersagen (100% lokal, CPU-only)

Was macht das ML-System?

Das ML-System analysiert deine bisherigen Trainingseinheiten und erstellt Gewichtsvorhersagen für deine nächsten Sätze. Je mehr du trainierst, desto präziser werden die Vorhersagen.

Wo wird es verwendet?

Im Training siehst du "Empfehlung: X kg × Y Wdh" - das kommt vom ML-System!

Was ist R² Score?

Zeigt wie genau das Modell vorhersagt. 1.000 = perfekt, >0.7 = gut, <0.5 = noch zu wenig Daten

Was ist MAE (kg)?

Durchschnittlicher Vorhersagefehler in kg. Je niedriger, desto besser! <2.5kg = exzellent

100% privat: Läuft komplett auf deinem Server, keine Cloud, keine API-Calls, keine Kosten. CPU-only: Benötigt keine GPU, Training dauert nur 2-5 Sekunden.
